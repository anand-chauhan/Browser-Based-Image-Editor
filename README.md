## Browser-Based Image Editor

Link : https://browser-based-image-editor-rouge.vercel.app/

A lightweight and interactive browser-based image editor built using JavaScript and HTML5 Canvas. This application allows users to upload images, apply real-time 
visual effects using CSS filters, and download the edited image instantly.
The project provides full control over image adjustments such as brightness, contrast, saturation, hue rotation, blur, grayscale, sepia, opacity, and invert. 
It also includes pre-defined filter presets like Drama, Vintage, Old School, Cool, Warm, and more to enhance images with a single click.
Designed with a dynamic UI, all filters are generated programmatically, making the application scalable and easy to extend. This project demonstrates practical 
use of Canvas API, DOM manipulation, and real-time image processing in the browser without any external libraries.

🚀 Features

📤 Upload image from local system
🎛️ Adjustable filters (brightness, contrast, saturation, etc.)
🎨 Predefined presets (Drama, Vintage, Retro, Black & White, etc.)
🔄 Reset filters to default
💾 Download edited image
⚡ Real-time preview using Canvas API
🧩 Dynamic UI generation for filters

🛠️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
Canvas API
